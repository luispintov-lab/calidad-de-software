Inicio Documentación Equipo

Descripción del Proyecto

Este proyecto tiene como objetivo aplicar los principios de calidad de software mediante el desarrollo de una aplicación web funcional, manteniendo las mejores prácticas de diseño y validación.

Funcionalidades

  • Registro de usuarios
  • Control de versiones con Git
  • Validaciones automáticas
